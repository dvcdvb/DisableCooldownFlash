# Disable Cooldown Flash

![Lua](https://img.shields.io/badge/lua-%232C2D72.svg?style=for-the-badge&logo=lua&logoColor=white)

Disable Cooldown Flash is a lightweight World of Warcraft addon that removes the Blizzard cooldown ready flash (bling) from action buttons.

## Features

- Disables the default Blizzard cooldown ready flash.
- Works with the default UI and action bar addons such as Bartender4.
- Lightweight with no configuration required.
- No external dependencies.

## Installation

1. Download or clone this repository.
2. Copy the `DisableCooldownFlash` folder into:

   ```
   World of Warcraft/_retail_/Interface/AddOns/
   ```

3. Restart World of Warcraft or type:

   ```
   /reload
   ```

## Compatibility

- World of Warcraft Retail 12.0.7
